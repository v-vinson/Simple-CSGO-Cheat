## 中文文档 | Chinese Documentation

### 项目介绍
一款简易的CSGO辅助工具，提供基础的自动开枪（Trigger）和锁头射击（Aimbot）功能，仅用于技术学习与研究。

### 依赖环境
通过 pip 安装所需依赖包：
- `pip install pymem`
- `pip install pynput`

### 使用说明
1. 运行项目根目录下的 `main.py` 启动程序。
2. 功能触发按键（二选一即可）：
   - 自动开枪：鼠标中键（Mouse3）或 `C` 键
   - 锁头射击：鼠标后侧键（Mouse4）或左 Alt 键
3. 按下 `End` 键即可关闭程序。

### 注意事项
- 本项目仅适配 CSGO 原版，未针对 CS2 进行适配。
- 停止维护后，不再提供漏洞修复、功能更新及兼容性调整。

## English Documentation

### Project Description
A simple auxiliary tool for CSGO, providing basic Trigger (auto-fire) and Aimbot (headshot lock) functions. For technical learning and research purposes only.

### Dependencies
Install required packages via pip:
- `pip install pymem`
- `pip install pynput`

### Usage
1. Run `main.py` in the project root directory to start the program.
2. Function trigger keys (either is acceptable):
   - Trigger: Mouse3 (middle mouse button) or `C` key
   - Aimbot: Mouse4 (side mouse button) or Left Alt key
3. Press the `End` key to close the program.

### Notes
- This project is only compatible with the original CSGO, not adapted for CS2.

- Game version updates may cause the tool to fail or behave abnormally.
